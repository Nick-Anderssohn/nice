# Security Policy

## Supported versions

Only the latest released version of Nice is actively maintained. Security fixes are not backported to earlier releases.

| Version | Supported |
|---------|-----------|
| latest  | yes       |
| older   | no        |

## Reporting a vulnerability

**Please do not report security issues in public GitHub issues.**

Report vulnerabilities privately using GitHub's built-in security advisory feature:

**[Submit a private advisory →](https://github.com/Nick-Anderssohn/nice/security/advisories/new)**

You'll receive a response within a week. Once a fix is ready, we'll coordinate disclosure timing with you before publishing.
